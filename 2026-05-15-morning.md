## 1. Anthropic 的 9000 亿美元时刻

5月12日，Bloomberg 和 Forbes 先后确认：Anthropic 正就以超 9000 亿美元投前估值，筹集至少 300 亿美元新一轮融资。（来源：Bloomberg, Forbes, 36氪）

几个数字说明一切：

- 估值 9000 亿美元——超越 OpenAI，成为全球估值最高的未上市公司
- 融资 300 亿美元——AI 史上最大单轮融资
- 一个月前（4月7日）Anthropic 刚宣布年营收 run rate 突破 300 亿美元，与 Google/Broadcom 签下 3.5GW TPU 算力协议
- 本轮资金将用于算力基建和 2026 年下半年可能的 IPO（来源：Bloomberg）

从去年被 OpenAI 甩开半个身位，到今年估值反超——Anthropic 的路径是：不做聊天产品、死磕企业级 API + Claude Code + 安全定位。（来源：Forbes）

**编辑点评：** 9000 亿美元买的是什么？不是 Claude 的聊天能力，而是"Claude 正在吃掉企业软件的工作流"。从 Claude Code 到 MCP 协议到 Agent 调用层，Anthropic 押注的是 AI 原生工作方式的底层话语权。

## 2. 具身智能「资本核爆」：前4月融资超 550 亿

2026 年上半年最火赛道——具身智能行业前 4 月融资超 200 笔，总规模超 550 亿元。最大单笔来自它石智航：4 月完成 4.55 亿美元 Pre-A 轮，刷新纪录。（来源：36氪、第一财经、澎湃新闻）

**编辑点评：** 大模型还没完全落地，资本已经在抢押"AI 的下一站"了。具身智能的硬件门槛注定了洗牌会比大模型更血腥。

## 3. 前大疆核心成员做消费级 CNC，陆奇也投了

前大疆核心成员创办的消费级 CNC 项目获美团、昆仑资本、奇绩创坛（陆奇旗下）投资近亿元。定位"桌面级精密加工"。（来源：36氪）

**编辑点评：** 大疆系创业图谱又添一员。从无人机到 CNC，从天空到桌面——中国硬件创业者的赛道切换速度令人瞩目。但消费级 CNC 是否成立，关键看"为什么不是你买 3D 打印机"这个问题的答案。
