# AI科技速递 | 2026年5月15日

> 早上好。今天三条主线：具身智能开年狂揽550亿融资、OpenAI ChatGPT换脑、Anthropic年营收冲300亿美元。

---

## 1. 具身智能上演「资本核爆」：前4月融资超550亿元

如果2025年的关键词是「大模型」，那2026年上半年毫无疑问属于「具身智能」。

据36氪报道，2026年上半年最火赛道——具身智能行业前4月融资超200笔，总规模超550亿元。同期，据第一财经不完全统计，仅截至4月10日，国内具身智能领域就已发生269起融资事件，其中122起披露具体金额，合计约345亿元。平均每天2.6起融资，每天超过3亿元资本涌入。（来源：36氪、第一财经、澎湃新闻）

其中最大一笔来自它石智航——2026年4月完成4.55亿美元（约合人民币33亿元）Pre-A轮融资，刷新了中国具身智能有史以来最大单轮融资纪录。（来源：腾讯新闻）

英诺科创基金合伙人王晟向数智前线感慨："哪怕是移动互联网最疯狂的时候，我们也没有见过这种节奏。"（来源：澎湃新闻）

**编辑点评：** 具身智能正在复现2023-2024年大模型的资本轨迹。区别在于，大模型更像"软件层"革命，而具身智能涉及硬件、供应链、场景落地，试错成本高一个数量级。资本密度越高，洗牌也会来得越快。

## 2. OpenAI 发布 GPT-5.5 Instant：ChatGPT 默认模型换新脑

5月5日，OpenAI 正式发布 GPT-5.5 Instant，取代 GPT-5.3 Instant 成为 ChatGPT 的默认模型。这是继4月23日 GPT-5.5 Thinking 和 GPT-5.5 Pro 发布后，该系列面向大众用户的"日常版"。

据 TechCrunch 报道，OpenAI 称新模型在减少幻觉方面有明显改进，同时提升了推理速度和个性化控制能力。GPT-5.5 Instant 对所有 ChatGPT 用户免费开放，而 Thinking 和 Pro 版本则需要订阅。（来源：TechCrunch、OpenAI 官方博客）

与此同时，OpenAI 上月还发布了 GPT-Rosalind——面向生命科学领域的前沿推理模型，专注于药物发现、蛋白质工程和基因组学研究，以科学家 Rosalind Franklin 命名。（来源：Reuters、OpenAI 官方）

**编辑点评：** OpenAI 的产品线分层越来越清晰——Rosalind 打科学垂直、Thinking/Pro 打高端订阅、Instant 守住免费基本盘。但「减少幻觉」这类表述已经听了三年，用户需要的是可量化的进步。

## 3. Anthropic 年营收突破300亿美元，签下史上最大算力协议

4月7日，Anthropic 宣布年化营收（run rate）突破300亿美元，同时与 Google 和 Broadcom 签署了史无前例的算力合作协议——3.5GW 的下一代 TPU 算力，预计2027年起陆续上线。（来源：TechCrunch、The Next Web、Tom's Hardware）

这笔交易是 Anthropic 历史上最大规模的算力承诺。该公司目前同时在 AWS Trainium、Google TPU 和 NVIDIA GPU 三种硬件上训练和运行 Claude 模型，CEO Dario Amodei 表示此举是为了"将工作负载匹配到最适合的芯片"。（来源：Anthropic 官方博客）

**编辑点评：** 300亿美元 run rate 意味着 Anthropic 已经从"挑战者"变成 OpenAI 的正面竞争者。而 3.5GW 算力是什么概念？大约相当于一个小型城市的电力需求。AI 竞赛的军备逻辑已经从"买多少 GPU"变成了"建多大的发电站"。
